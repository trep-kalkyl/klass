Usage Examples

Basic Switch

<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" role="switch" id="sw1">
  <label class="form-check-label" for="sw1">Notifications</label>
</div>

Switch with State Text (Module)

<div class="form-check form-switch switch-with-state">
  <input class="form-check-input" type="checkbox" role="switch" id="sw1" checked>
  <label class="form-check-label" for="sw1">
    Notifications <span class="switch-state">(On)</span>
  </label>
</div>
