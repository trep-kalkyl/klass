KLASS Alert Guide

This guide shows how to use KLASS alert components for feedback messages and notifications.

Basic Alerts

Standard alert types for success, error, warning and info.

Success!Your changes have been saved.
Error!Could not complete the action.
Warning!You have unsaved changes.
Info:The system will restart at 22:00.

Dismissible Alerts

Alerts with a close button.

This is a message you can close.

Alert with More Content

Alerts can contain headings, paragraphs, links and separators.

Well done!

You have completed all steps in the configuration. Your system is now fully operational and all settings have been saved to the database.

You can proceed by clicking this link.
